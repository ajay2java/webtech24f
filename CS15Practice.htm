Picture Gallery with JavaScript

The page below holds a table that has 2 rows and 2 columns, each cell holding an image. Think of this as your thumb-nail view. The table has been floated left. There is another image to the right of the table, that is currently empty and hence you cannot see it. It has an ID of "big" and we will refer to this as the big image. Below the big image is a caption, also empty, and invisible. Write the functionality in Javascript whereby, if you click on a thumb-nail image, that image is shown as a big image, much like how a picture gallery will work. Make up a different caption for each image and show that caption below the image. Hint: this would require the use of "this.id" as well as the use of conditional statements.
