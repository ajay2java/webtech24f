SpongeBob Character Design:

  1. Paste a transparent traceable picture from the web

    2. This image will be switched beween visible & invisible, to be simply used as a benchmark for the actual character we will be drawing.

  2. Use the 4th tool from the top of the picture to start setting points at the edges

    3. Setting points on straight surfaces around the image, will allow you to modify the lines between the points to curve, conforming to the curves in the benchmark image.

  3. Use the color fill tool (13th from top) to have drawn shapes result in the same color!

    4. We can extract the color from the becnhmark image into the shapes we draw, making our drawing look more like the image.

  4. Use the draw circle feature for the dots!

    5. SpongeBob has dots, so we can use circles to create the dots and modify the shape with the second control from the top. This will make the dots more uniform.

  5. Repeat the same with the other dots!

  6. Continue same tracing & dot drawing for SpongeBob's side as we did with the face!

  7. Trace the arms from the benchmark image!

    8. Here, we essentially complete the same for the arms as we did for the face, filling in the colors baased on the benchmark image.

  8. We can duplicate & reflect the right arm to get the left arm!!

    9. Drawing the arms can be hard...but we can simply make it eaier by drawing one, and then duplicating & reflecting it to become the other! Key: Make sure to rename the duplicate one under the layers section.

  9. Make sure to organize the layers based on how you see fit!

    10. For example, in the image, it seems that the left arm is behind the face. Well, this is by design as we put the left arm layer behind the face layer! This, my friends, is the beauty of drawing a character part-by-part!

  10. Continue tracing & organizing the layers!

    11. And...WOWWWW!!! We have our SpongeBob character! It came together so smoothly!!!
